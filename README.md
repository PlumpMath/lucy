# Lucy

A DSL for writing CYPHER query like Clojure, which interacts with Neo4J via `clojure.java.jdbc` and the JDBC Neo4J driver.

## Installation

Include in your `project.clj` file:

	:dependencies [[lucy "0.1.0-SNAPSHOT"]]
